Docs: https://docs.movementnetwork.xyz/devs/movementcli

## Tutorial Structure

This is a tutorial to learn Move Smart Contract Language for beginners. This tutorial implements a stabletoken step by step, with an explanation of each step including related tests. The tutorial consists of 4 chapters:

- Volume 1: Basic Syntax & Move Logic
- Volume 2: Treasury Capability & Intermediate Testing
- Volume 3: Creating & Integrating a token
- Volume 4: Integrating Oracle Price Feed
